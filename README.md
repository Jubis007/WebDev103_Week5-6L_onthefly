## Debugging a full stack application sometimes feels like trying to navigate a foreign airport without a map, but I finally landed the plane! ✈️

I am thrilled to share my latest full stack application, On the Fly. I wanted to build a comprehensive travel planning app from scratch to challenge myself with connecting a modern user interface to a robust backend database.

Connecting a sleek client side experience with complex server side data management was an incredibly rewarding process. Here is a breakdown of what I built under the hood:

* Designed a relational PostgreSQL database deployed on Render, complete with a join table to link trips and destinations.
* Developed a customized backend using Node.js and Express to handle RESTful API requests.
* Built a dynamic React frontend utilizing Vite and modern React Hooks to seamlessly manage state and update the UI.
* Implemented full CRUD functionality, allowing users to create itineraries, vote on activities, and link available destinations.
